# GPT2 fine-tuning to generate styled English texts (bios, lyrics, etc)

Automatically fine-tune gpt2 to your use case (must be in English)

# Prerequisite

Have the following libraries installed:

1. pandas
2. numpy
3. langdetect
4. torch
5. fastai
